# 현대 영어에서의 고전 복수형과 규칙 복수형의 경쟁 메커니즘

**Google Ngram과 COCA 데이터를 중심으로 한 데이터 분석 기반 현상 조사**

라틴·그리스어 파생 명사의 복수형이 영어식 규칙 복수형(-s/-es)으로 고착되거나, 고전 복수형으로 유지되거나, 두 형태가 공존하는 양상을 **Google Ngram**(통시적 사용량·연어)과 **COCA**(공시적 맥락)로 분석한 코퍼스 기반 연구다. 12개 명사쌍을 **3층위 분석 틀**로 분절하여, 단순한 형태 경쟁이 아니라 의미·레지스터·제도·역사가 함께 작용한 결과임을 밝힌다.

> 저자: 오신영 · 김동현 · 정희연 · 이윤재

## 분석 틀: 3층위 구조

각 사례를 다음 세 층위로 **분절**하고, 사례 안에서 **위계**를 세운다.

| 층위 | 질문 | 사용 도구 |
| --- | --- | --- |
| **1차 — 현재 사용 상태** | 지금 어느 형태가 우세한가? 상승·역전·공존은 언제? | Google Ngram **사용량 그래프** |
| **2차 — 변화의 속도·방향** | 그 우세가 어떻게 형성되었나? (장기 유지 / 우세 전환 / 장기 경쟁 / 빠른 수렴) | Google Ngram **사용량 그래프**(경로 해석) |
| **3차 — 작동 메커니즘** | 왜 그 방향으로 변했나? (의미 분화 / register 분화 / 경쟁) | Google Ngram **연어 그래프** + **COCA 맥락 분석** |

> **읽는 순서**: 각 사례 문서는 요청에 따라 **① 결론(메커니즘) → ② 연구 결과(3층위 요약) → ③ 과정 및 결론 도달 과정(도구) → ④ 세부 정보(1차/2차/3차 구간 분절 + 역사적 제언)** 순으로 배치되어 있다.

> **이미지 처리**: 각 사례 본문에는 대표 이미지(Ngram 사용량 그래프) 1개만 두고, 형용사·명사 연어 그래프와 COCA 맥락 캡처는 각 문서의 **참조 이미지** 절에 분리했다.
