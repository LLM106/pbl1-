# 참고문헌 및 사용 도구

## 사용 도구 (Tools)

본 연구의 모든 데이터는 아래 두 코퍼스 도구로 수집·분석되었다. 각 도구가 어느 층위에서 어떻게 쓰였는지는 [방법론](methodology.md#ⅳ-3층위-분석-틀과-도구-사용)에 상술되어 있다.

### Google Books Ngram Viewer
- **출처**: <https://books.google.com/ngrams>
- **용도**:
  - **1차 (현재 사용 상태)** — 각 복수형 쌍의 장기 사용량 추세, 우세 형태, 상승 시점, 역전·공존 구간 확인.
  - **2차 (변화의 속도·방향)** — 동일 사용량 그래프에서 우세 형성 경로(장기 유지 / 우세 전환 / 장기 경쟁 / 빠른 수렴) 해석.
  - **3차 (작동 메커니즘)** — 각 형태 앞에 오는 **형용사·명사 연어** 그래프로 결합 양상 분석.

### COCA — Corpus of Contemporary American English
- **출처**: <https://www.english-corpora.org/coca/>
- **용도**:
  - **3차 (작동 메커니즘)** — 각 복수형의 실제 용례를 ACAD/NEWS/SPOK/MAG/FIC/WEB/BLOG 등 매체(register)별로 검토하여, 두 형태가 동일 의미 영역에서 경쟁하는지, 의미 기능을 분담하는지, 특정 레지스터·담화 공동체와 연결되는지 해석.

## 규범 자료 (사전·스타일 가이드)

- **Oxford English Dictionary (OED)** — 복수형 표제·용법 구분.
- **Merriam-Webster Dictionary** — 책 찾아보기에는 *indexes*, 수학·과학·경제 지표에는 *indices* 구분.
- **Fowler, *A Dictionary of Modern English Usage*** — 형태별 사용 차이 명시.

> 위 규범 자료는 특히 `indices/indexes` 사례에서 의미 분화가 제도적 규범으로 강화되었음을 보이는 근거로 활용되었다.

## 역사적 맥락 참고 영역

각 사례의 **역사적 제언**은 해당 형태의 규범화·표준화 과정(예: 식물학의 라틴어 복수형 표준화, 수학에서 *matrix*의 라틴어 복수형 규범화, 금융경제학에서 *premia* 재부상 등)을 근거로 한다. 세부 내용은 각 사례 문서의 `4-4. 역사적 제언` 절을 참고.

> 참고: 역사적 맥락을 뒷받침하는 학술 논문 출처는 별도 보강 대상이다(원 보고서의 "최종 수정" 메모 중 *역사적 맥락 관련 논문 추가* 항목 참조).
